## Setup 
### Manual install
1. Clone this repository
```
git clone https://github.com/RedLordezh7Venom/VPeakFit.git
cd VPeakFit
```
2. Install Requirements
```
npm install
pip install -r requirements.txt
```
3. Run the program
In the same terminal type:
```
npm run dev
python app.py
```
